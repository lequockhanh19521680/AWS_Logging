## Libraries
- `@ngx-translate/core` + `@ngx-translate/http-loader`
- Works with Angular 21 Standalone Components.

## Directory
- Translation files per app:
  - `client/web-portal/src/assets/i18n/en.json`, `vi.json`
  - `client/back-office/src/assets/i18n/en.json`, `vi.json`

## Providers
- Use `importProvidersFrom(TranslateModule.forRoot(...))` in `app.config.ts`.
- Loader: `TranslateHttpLoader` with base `/assets/i18n/`.

## Language Management
- Persist current language in `localStorage` (`lang` key).
- Default to `en`, allow switching with a simple `LanguageService`.
- Attach `Accept-Language` in HTTP interceptor.

## Template Usage
- Replace static labels with `{{ 'key' | translate }}`.
- Example keys:
  - `login.title`, `login.username`, `login.password`, `login.submit`
  - `otp.title`, `otp.input`, `otp.submit`

## SSR Note
- Client-side translation is acceptable; server can ignore `Accept-Language` if not rendering on the server.

## Adding a Language
- Create `<lang>.json` under `assets/i18n/`.
- Add keys consistent with existing namespaces.
- Call `LanguageService.use('<lang>')`.
